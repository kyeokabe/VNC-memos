**Environments:**  
Macbook Pro 15' Mid 2012  
OSX 10.11.3  
TensorFlow r0.8 (OSX, CPU only)  
Google Chrome ver. 50.0.2661.86 (64-bit)  
  
open terminal
enter:  
```
cd <TF_2LayerNet directory>  
python TF_2LayerNet.py  
```

```

```

open a Chrome browser  
in the address bar, enter:  

```
http://0.0.0.0:6006  
```
