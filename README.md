##### Executive Summary of [FarmVNC](https://web.stanford.edu/group/farmshare/cgi-bin/wiki/index.php/FarmVNC)

**open a terminal on Mac** 
```
  ssh <SUID>@corn.stanford.edu
  <enter SUID PW>
  <double authentication>
  vncpasswd
  module load farmvnc
  farmvnc
  farmvnc 1440x900
```
